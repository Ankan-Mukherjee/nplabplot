<h3> Loading data: </h3>
To open a folder containing data, please click open folder and browse to the relevant folder containg your data. Alternatively, use the drag and drop the feature to load a folder. Then, using the drop downs, select the X, Y, and Z axes.

<h3> Viewing data: </h3>
Use the scroll on the plot space to zoom in or out on the data. You can scroll on the X or Y axes lines to zoom in/out to zoom in only on that axis. Use the easel icon at the bottom left corner to reset zoom. Alternatively, right click and select View All. For 1-D data, use the color dropdown to select the plot line color and the linestyle drop down to select the line style. For 2-D plots, right scroll on the histogram to adjust range, use the mouse left click to drag and adjust sliders. Right click to change colormap or to change scale to log scale.

<h3> Exporting data/plot <h3>
Right click and select export. Select the appropriate option and click Export. Save in your preferred directory with your preferred filename. For saving image, select entire scene. Note that it saves the image as it shows on screen when clicking export.
